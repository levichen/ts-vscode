
## What is this?

The skeleton for TypeScript project. 

## How to use?

### Build you project
``` npm run build ```

### Check your coding style with tslint
``` npm run lint ```

### Test your code
``` npm run test ```

###  Automatically build your code when you are developing
``` npm run build -- --watch ``` 

###  Automatically run your test code when you are developing
``` npm run test -- --watch ```
